    # About

    # Installation

    ## Via NPM/PNPM/Yarn

    Install the packages:

```bash
npm install @esnet/packets-ui-css @esnet/packets-ui-web
```

    Import in your JavaScript/TypeScript entry point:

```javascript
import '@esnet/packets-ui-web';
```

    Import the CSS:

```css
@import "@esnet/packets-ui-css/dist/styles.css";
```

    ## Via CDN (No Build System)

    For projects without a build system, load from unpkg:

```html
<!DOCTYPE html>
<html>
<head>
  <!-- Packets CSS -->
  <link rel="stylesheet" href="https://unpkg.com/@esnet/packets-ui-css@latest/dist/styles.css" />

  <!-- Fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link href="https://fonts.googleapis.com/css2?family=Signika:wght@300..700&family=Source+Code+Pro:ital,wght@0,200..900;1,200..900&family=Source+Sans+3:ital,wght@0,200..900;1,200..900&display=swap" rel="stylesheet" />
</head>
<body class="packets">
  <pkts-button variant="primary">Click Me</pkts-button>

  <!-- Load Web Components -->
  <script type="module" src="https://unpkg.com/@esnet/packets-ui-web@latest/dist/index.js"></script>
</body>
</html>
```

    # Usage

    Web Components work with any framework or vanilla HTML. Add the `packets` class to your root container:

```html
<div class="packets">
  <pkts-button variant="primary">Click Me</pkts-button>
  <pkts-input-text placeholder="Enter text..."></pkts-input-text>
  <pkts-alert variant="info" title="Info Alert">
    This is an informational message.
  </pkts-alert>
</div>
```

    ## With Vue

```vue
<template>
  <div class="packets">
    <pkts-button variant="primary">Click Me</pkts-button>
    <pkts-input-text placeholder="Enter text..."></pkts-input-text>
  </div>
</template>
```

    ## With Vanilla JavaScript

```javascript
const button = document.createElement('pkts-button');
button.setAttribute('variant', 'primary');
button.textContent = 'Click Me';
document.body.appendChild(button);
```


    # TypeScript Support

    Web Components include full TypeScript type definitions. Import the package and TypeScript will provide IntelliSense for all custom element properties and events:

```typescript
import '@esnet/packets-ui-web';

const button = document.querySelector('pkts-button');
if (button) {
  button.variant = 'primary'; // Fully typed
}
```
