# Packets Design System - CSS Components

Welcome to the CSS component documentation for the Packets Design System.

## About

This documentation showcases pure HTML/CSS implementations of Packets Design System components. Each component story shows the HTML markup and CSS classes needed to use the component in your project.

## Installation

To use these components in your project:

```bash
npm install @esnet/packets-ui-css
```

## Usage

1. Import the CSS in your project:

```css
@import "@esnet/packets-ui-css/dist/styles.css";
```

2. Add the `packets` class to your root container:

```html
<div class="packets">
  <!-- Your components here -->
</div>
```

3. Optionally, add `dark` or `light` classes to force a specific theme:

```html
<div class="packets dark">
  <!-- Dark mode components -->
</div>
```

## Fonts

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link href="https://fonts.googleapis.com/css2?family=Signika:wght@300..700&family=Source+Code+Pro:ital,wght@0,200..900;1,200..900&family=Source+Sans+3:ital,wght@0,200..900;1,200..900&family=Zilla+Slab:ital,wght@0,300;0,400;0,500;0,600;0,700;1,300;1,400;1,500;1,600;1,700&display=swap" rel="stylesheet">
```
