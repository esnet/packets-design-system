    # Installation

    ## Via NPM/PNPM/Yarn

    Install the package:

```bash
npm install @esnet/packets-ui-web
```

    Import in your JavaScript/TypeScript:

```javascript
import '@esnet/packets-ui-web';
```

    ## Via CDN (No Build System)

    For projects without a build system, you can use unpkg CDN:

```html
<!DOCTYPE html>
<html>
<head>
  <!-- Fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link href="https://fonts.googleapis.com/css2?family=Signika:wght@300..700&family=Source+Code+Pro:ital,wght@0,200..900;1,200..900&family=Source+Sans+3:ital,wght@0,200..900;1,200..900&display=swap" rel="stylesheet" />
</head>
<body class="packets">
  <!-- Your Web Components here -->
  <pkts-button variant="primary">Click Me</pkts-button>
  <pkts-input-text placeholder="Enter text..."></pkts-input-text>

  <!-- Load Web Components from unpkg -->
  <script type="module" src="https://unpkg.com/@esnet/packets-ui-web@latest/dist/index.js"></script>
</body>
</html>
```

    # Usage

    Web Components work with any framework or vanilla HTML:

```html
<div class="packets">
  <pkts-button variant="primary">Click Me</pkts-button>
  <pkts-input-text placeholder="Enter text..."></pkts-input-text>
  <pkts-alert variant="info" title="Info Alert">
    This is an informational message.
  </pkts-alert>
</div>
```

    ## With React

```tsx
function MyComponent() {
  return (
    <div className="packets">
      <pkts-button variant="primary">Click Me</pkts-button>
      <pkts-input-text placeholder="Enter text..."></pkts-input-text>
    </div>
  );
}
```

    ## With Vue

```vue
<template>
  <div class="packets">
    <pkts-button variant="primary">Click Me</pkts-button>
    <pkts-input-text placeholder="Enter text..."></pkts-input-text>
  </div>
</template>
```

    ## With Vanilla JavaScript

```javascript
const button = document.createElement('pkts-button');
button.setAttribute('variant', 'primary');
button.textContent = 'Click Me';
document.body.appendChild(button);
```

    # TypeScript Support

    Web Components include full TypeScript type definitions. When using TypeScript, you'll get full IntelliSense and type checking:

```typescript
import '@esnet/packets-ui-web';

// TypeScript knows about all custom element properties and methods
const button = document.querySelector('pkts-button');
if (button) {
  button.variant = 'primary'; // Fully typed
}
```
