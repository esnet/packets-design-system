# Datum

### Overview

Datum is a name-value pair visualization that displays a small label above a larger value. The label uses a secondary (alt) copy color to visually de-emphasize the field name, while the value uses the primary copy color for emphasis. When you need to show multiple related key-value pairs together, consider grouping several Datum components in a flex or grid layout.

### States and Behaviours

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - Surfacing a single key-value metadata field in a dashboard
- How to use this component
  - Do:
    - Use short, descriptive label text via the `title` prop (e.g. "Bandwidth", "Source IP")
    - Pass the value as `children` - this can be a string, number, or a simple inline element
  - Don't:
    - Use Datum for form fields or editable inputs
    - Use long paragraphs as the value, only concise data points
