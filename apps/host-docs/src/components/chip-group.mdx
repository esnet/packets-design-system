# Chip Group

### Overview

ChipGroup is a composable container that arranges a collection of `Chip` components in an inline, wrapping flex layout with consistent spacing. Use this component whenever you need to display multiple chips together, such as a set of applied filters.

### States and Behaviours

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - Displaying a collection of filter chips or tag selections as a cohesive group
  - Allowing the group layout to wrap naturally when chips exceed the available width
- How to use this component
  - Do:
    - Only inlcude `PktsChip` elements as children, other elements will be filtered out
    - Let the group wrap naturally rather than constraining it to a fixed single row
  - Don't:
    - Use ChipGroup for a single chip - render a standalone `Chip` instead
