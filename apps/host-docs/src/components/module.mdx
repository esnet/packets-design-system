# Module

### Overview

A generic section wrapper that applies consistent horizontal padding at different breakpoints. When the `surface` prop is enabled the component also adds a border radius and vertical padding to visually elevate the content as a card-like surface. Use this component to frame page sections or content blocks; prefer a more specific layout component (such as Card) when richer structural features like headers, footers, or actions are required.

### States and Behaviours

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - When wrapping a section of page content that needs standard horizontal gutters
  - When a lightweight visual container with optional surface elevation is needed without the overhead of a full card structure
- How to use this component
  - Do:
    - Use the `surface` prop to visually separate the module from its background when elevation is needed
    - Provide a `title` only when the section heading belongs semantically inside the module
  - Don't:
    - Use this component as a replacement for Card when actions, media, or structured header/footer areas are required
    - Nest multiple surface modules directly inside each other - use spacing tokens instead
