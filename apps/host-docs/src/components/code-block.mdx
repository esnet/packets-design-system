# Code Block

### Overview

CodeBlock renders a block of preformatted code text with a language chip label and a built-in copy-to-clipboard button. The copy button defaults to writing the code string to the system clipboard and briefly shows a checkmark confirmation. Use this component for displaying code snippets, configuration examples, or command-line instructions in documentation.

### States and Behaviours

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - Displaying code snippets, shell commands, or configuration examples that users may need to copy
  - Inline developer documentation where syntax context (via the language label) aids comprehension
- How to use this component
  - Do:
    - Pass a raw JavaScript template string as `children` to preserve whitespace and line breaks
    - Set the `language` prop to accurately label the code type (e.g. `"python"`, `"typescript"`)
  - Don't:
    - Pass React nodes or rich content as children, only strings
