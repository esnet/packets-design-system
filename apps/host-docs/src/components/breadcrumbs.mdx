# Breadcrumbs

### Overview

A horizontal navigation aid that displays the current page's location within a site hierarchy as a list of links separated by `/` dividers. The React component accepts an array of breadcrumb objects and an optional custom `renderLink` function, making it router-agnostic. The Web Component uses slotted `<li>` elements for flexibility. The last item in the list has its divider suppressed automatically.

### States and Behaviours

**Themes**: Light (default), Dark

**States**: Default, Hover

### Best Practices

- When to use this component
  - When the page is more than one level deep in a site hierarchy and users benefit from knowing their current location
  - When users may need to navigate up to a parent section without using the browser's back button
- How to use this component
  - Do:
    - List items in hierarchy order from root to current page, left to right
    - Keep breadcrumb labels short and matching the actual page titles they link to
  - Don't:
    - Use breadcrumbs on top-level or single-depth pages where location context adds no value
    - Duplicate the current page as a link - the last item should be informational only
