# Comma Seperated List

### Overview

CommaSeperatedList renders an array of items as an inline `<ul>` where each item is displayed without bullets and separated by a trailing comma - except for the last item. It is a purely presentational component with no interactive states.

### States and Behaviours

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - Showing comma separated values as an HTML list rather than using commas (better for copying)
- How to use this component
  - Do:
    - Keep items concise - long values in an inline list can make content hard to scan
  - Don't:
    - Use this component for ordered or hierarchical lists - use a standard `<ol>` or `<ul>` instead
    - Nest CommaSeperatedList inside another CommaSeperatedList
