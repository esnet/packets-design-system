# Form Section

### Overview

The Form Section component is a layout wrapper for groups of related form inputs. It organizes content into a two-column grid: a narrow left column for a section title and optional description, and a wider right column for the form controls.

### States and Behaviours

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - When a form contains multiple logical groups of inputs that benefit from a descriptive label column
- How to use this component
  - Do:
    - Provide a meaningful `title` to help users understand the purpose of the grouped inputs
  - Don't:
    - Nest Form Sections inside one another - keep groupings flat for clarity
    - Use this component for non-form content where a simpler layout utility would suffice
