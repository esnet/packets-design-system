# Divider

### Overview

The Divider is a styled HTML `<hr>` element used to visually separate sections of content, rendering as a thin horizontal line. Prefer semantic section structure or spacing utilities when a visual separator is not strictly needed.

### States and Behaviours

**Variants**: primary, branded

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - Between logically distinct sections of a page or form
  - As a lightweight visual cue when additional whitespace alone is insufficient
- How to use this component
  - Do:
    - Use the default (primary) variant for general-purpose separation
    - Use the branded variant sparingly to draw attention to a meaningful section boundary
  - Don't:
    - Use a divider as a substitute for proper heading hierarchy or spacing
    - Stack multiple dividers in close proximity
