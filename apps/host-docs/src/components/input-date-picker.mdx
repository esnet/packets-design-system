# Input Date Picker

### Overview

A popup calendar and time-wheel picker that supports single-date, time-only, datetime, and date-range selection modes. It is not recommended to use this component directly unless absolutely necessary - prefer `InputDate` for single-date selection or `InputDateRange` for range selection, both of which wrap this component and provide an associated text input box.

### States and Behaviours

**Themes**: Light (default), Dark

**States**: Default, Hover, Focus, Active, Disabled

### Best Practices

- When to use this component
  - When you need to embed a calendar or time picker inline and are building a higher-level input wrapper
  - When none of the higher-level date input components (`InputDate`, `InputDateRange`) suit your layout
- How to use this component
  - Do:
    - Use `InputDate` or `InputDateRange` in most cases instead of this component directly
    - Supply `dateSettings.min` and `dateSettings.max` to prevent selection of invalid dates
    - Use `type="daterange"` together with both `value`/`onChange` and `rangeEndValue`/`onChangeRangeEnd`
  - Don't:
    - Mount this component without a positioned ancestor - it is absolutely positioned and will overflow otherwise
    - Use `type="datetime"` when only date or only time is needed
