# Table Of Contents

### Overview

PktsTableOfContents renders a navigational list of anchor links representing the sections of a page. It supports an optional sticky positioning mode that keeps the component fixed at the top of the viewport as the user scrolls. Pass a list of link objects and an optional custom render function to control how each section link is displayed.

### States and Behaviours

**Themes**: Light (default), Dark

### Best Practices

- When to use this component
  - Use alongside long-form content pages where readers benefit from quick in-page navigation
  - Use when the page has three or more distinct sections that warrant direct links
- How to use this component
  - Do:
    - Enable `isSticky` when placing the component in a sidebar column so it follows the user as they scroll
    - Provide meaningful, concise section labels so links remain scannable
  - Don't:
    - Don't use on short pages with only one or two sections where a full table of contents adds unnecessary noise
    - Don't place inside a scrollable container if `isSticky` is enabled, as sticky positioning requires a scrolling ancestor
