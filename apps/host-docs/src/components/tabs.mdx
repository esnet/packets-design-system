# Tabs

### Overview

PktsTabs is a composable container that manages the layout of a horizontal list of tab items. It accepts PktsTab children and filters out any non-tab elements automatically. Optional props control whether a border is rendered around the tab strip and whether vertical padding is applied. Use PktsTab children to define each individual tab, and manage active state externally by setting the `isActive` prop on the appropriate tab.

### States and Behaviours

**Themes**: Light (default), Dark

**States**: Default, Hover, Active

### Best Practices

- When to use this component
  - Use to switch between related views or content panels within the same page context
  - Use when there are two or more distinct content areas that users switch between without navigating away
- How to use this component
  - Do:
    - Control active tab state externally by tracking a selected index and passing `isActive` to the relevant PktsTab
    - Use `border` when the tab strip needs visual separation from surrounding content
  - Don't:
    - Don't nest PktsTabs inside another PktsTabs — use a flat structure instead
    - Don't place non-tab elements as direct children, as they will be filtered out by the component
