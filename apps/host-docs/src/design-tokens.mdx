    # Design Tokens

    **Advanced Integration**

    Design tokens are the foundational design decisions of the Packets Design System, represented as data. They include colors, spacing, typography, motion, and more.

    ## When to Use Design Tokens

    Design tokens are for **advanced integrations** where you need to:

    - **Integrate with existing design systems** - You already have a design system (like Tailwind CSS, Material UI, etc.) and want to adopt Packets design values
    - **Build custom components** - Create your own components that match Packets design language
    - **Use a different framework or technology** - You're not using React, Web Components, or pure CSS components, but want to use Packets design values
    - **Customize or extend the design system** - You need to modify or extend Packets for your specific needs
    - **Integrate with design tools** - Use Packets tokens in Figma, Sketch, or other design tools

    ## Example Use Cases

    ### Integrating with Tailwind CSS

    If you have an existing project using Tailwind CSS, you can import Packets design tokens to customize your Tailwind configuration:

```javascript
// tailwind.config.js
import tokens from '@esnet/esnet-tokens/tokens.json';

export default {
  theme: {
    extend: {
      colors: {
        primary: tokens.color.primary.base.value,
        secondary: tokens.color.secondary.base.value,
        // ... more token mappings
      },
      spacing: {
        // Map Packets spacing tokens to Tailwind spacing
        xs: tokens.spacing.xs.value,
        sm: tokens.spacing.sm.value,
        // ... more spacing tokens
      },
    },
  },
};
```

    ### Building Custom Components

    Use design tokens directly in your custom components:

```typescript
import tokens from '@esnet/esnet-tokens/tokens.json';

const CustomButton = styled.button`
  background-color: ${tokens.color.primary.base.value};
  padding: ${tokens.spacing.md.value} ${tokens.spacing.lg.value};
  border-radius: ${tokens.radius.md.value};
  transition: all ${tokens.motion.duration.fast.value} ${tokens.motion.easing.ease.value};
`;
```
