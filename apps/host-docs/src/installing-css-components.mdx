    # Installation

    ## Via NPM/PNPM/Yarn

    Install the package:

```bash
npm install @esnet/packets-ui-css
```

    Import the CSS in your project:

```css
@import "@esnet/packets-ui-css/dist/styles.css";
```

    ## Via CDN (No Build System)

    For projects without a build system, you can use unpkg CDN:

```html
<!DOCTYPE html>
<html>
<head>
  <!-- Packets CSS via unpkg -->
  <link rel="stylesheet" href="https://unpkg.com/@esnet/packets-ui-css@latest/dist/styles.css" />

  <!-- Fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link href="https://fonts.googleapis.com/css2?family=Signika:wght@300..700&family=Source+Code+Pro:ital,wght@0,200..900;1,200..900&family=Source+Sans+3:ital,wght@0,200..900;1,200..900&display=swap" rel="stylesheet" />
</head>
<body class="packets">
  <!-- Your HTML with Packets CSS classes -->
  <button class="pkts-button pkts-primary">Click Me</button>
</body>
</html>
```

  </div>
</div>

    # Usage

    1. Add the `packets` class to your root container:

```html
<div class="packets">
  <!-- Your components here -->
</div>
```

    2. Use component classes as shown in the examples:

```html
<div class="packets">
  <button class="pkts-button pkts-primary">Primary Button</button>

  <div class="pkts-input-text">
    <input type="text" placeholder="Enter text...">
  </div>

  <div class="pkts-alert pkts-info">
    <aside class="icon">
      <!-- Icon SVG here -->
    </aside>
    <div class="content">
      <strong>Info Alert</strong>
      <p>This is an informational message.</p>
    </div>
  </div>
</div>
```

    3. Optionally, add `dark` or `light` classes to force a specific theme:

```html
<div class="packets dark">
  <!-- Dark mode components -->
</div>
```
